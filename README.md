# Kawerify Tech Tournaments

A professional soccer tournament management platform built for gaming enthusiasts.

## About

Kawerify Tech Tournaments is a comprehensive web application designed to manage cup-style knockout soccer tournaments. The platform features automatic fixture shuffling, real-time leaderboards, player management, and a complete admin dashboard.

## Features

- **Tournament Management**: Cup-style knockout format with automatic fixture generation
- **Real-time Leaderboard**: Live rankings and statistics
- **Player Profiles**: Individual player stats and favorite teams
- **Admin Dashboard**: Complete tournament administration tools
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```sh
git clone <repository-url>
cd kawerify-tech-tournaments
```

2. Install dependencies:
```sh
npm install
```

3. Start the development server:
```sh
npm run dev
```

4. Open your browser and navigate to `http://localhost:8080`

### Building for Production

```sh
npm run build
```
